# Hi, I'm Steban_CL 🦝

Soy alguien que disfruta desarmar problemas para entender cómo funcionan por dentro. Mi camino en el desarrollo de software comenzó por la curiosidad de construir cosas desde cero, y hoy esa curiosidad se ha transformado en un enfoque basado en la **lógica y el análisis**.

No me considero un experto en la sintaxis de cada lenguaje, sino un arquitecto de soluciones. Mi metodología es el **Vibecoding**: utilizo la IA como un colaborador para manejar la escritura de código mientras yo me aseguro de que la lógica, el flujo de datos y la estructura del proyecto sean sólidos y funcionales.



## 👀 Intereses & Visión
- **Análisis de Sistemas:** Me gusta la fase previa al código; entender los requisitos y diseñar cómo se conectarán las piezas.
- **Vibecoding:** Exploro constantemente cómo la inteligencia artificial puede potenciar la creatividad y la velocidad de desarrollo.
- **Interactividad Dinámica:** Me atrae crear sitios donde el usuario sienta que la página responde de forma inteligente a sus acciones.
- **Evolución Constante:** Mi meta es integrar cada vez mejor el Backend con el Frontend, manteniendo siempre la simplicidad y la eficiencia.

## 🌱 Stack Tecnológico & Lógica
Este es el ecosistema de herramientas que utilizo para dar vida a mis ideas:

* **Lógica de Interfaz:** HTML5, CSS3 y Bootstrap 5 para estructuras limpias; JavaScript para la manipulación dinámica del DOM.
* **Lógica de Servidor:** PHP y Python, enfocados en entender el comportamiento del backend y el manejo de datos.
* **Modelado de Datos:** Diseño de bases de datos MySQL y análisis de modelos Entidad-Relación (MER).
* **Análisis Técnico:** Creación de diagramas de clase y elicitación de requisitos (ADSO).

## 🎯 Sobre mi proceso
Para mí, programar es una forma de organizar el pensamiento. Me gusta dedicar tiempo a reflexionar sobre mis proyectos y sobre cómo mejorar lo que ya he construido. Creo firmemente que la consistencia y la curiosidad son las que marcan la diferencia entre alguien que solo escribe código y alguien que construye software con sentido.

Actualmente, mi portafolio **v1.0** es el reflejo de este camino: un espacio donde la lógica de traducción y la estructura visual se unen para mostrar mi progreso como futuro desarrollador Full Stack.
